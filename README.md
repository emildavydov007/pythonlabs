# Лабораторная работа №1

## Задание
- Создайте репозиторий для дисциплины на GitHub или GitLab
- Склонируйте его себе на ПК
- Скачайте архив и распакуйте его в свой репозиторий. В нём 11 заданий, которые вам нужно выполнить
- Напишите отчёт в README.md.
- Сделайте commit и push

## Описание проделанной работы

В ходе выполнения работы были решены следующие задачи:

 **Решение задач:**
   - `00_distance.py`: Вычисление расстояний между городами по координатам с использованием словарей и формул.
   - `01_circle.py`: Расчёт площади круга и проверка попадания точек внутрь круга.
   - `02_operations.py`: Поиск арифметической комбинации для получения числа 25 из последовательности `1 2 3 4 5`.
   - `03_favorite_movies.py`: Извлечение названий фильмов из строки с помощью срезов.
   - `04_my_family.py`: Работа со списками: хранение членов семьи и их роста, вычисление сумм.
   - `05_zoo.py`: Манипуляции со списками: вставка, расширение, удаление, поиск индексов.
   - `06_songs_list.py`: Вычисление общего времени звучания песен из списка и словаря.
   - `07_secret.py`: Расшифровка сообщения с использованием сложных срезов строк.
   - `08_garden.py`: Операции с множествами: объединение, пересечение, разность.
   - `09_shopping.py`: Формирование нового словаря на основе исходных данных о магазинах.
   - `10_store.py`: Расчёт стоимости товаров на складе без использования циклов.

## Шпаргалка по работе с командами `git`

| Команда | Описание |
| :--- | :--- |
| `git init` | Инициализация нового репозитория в текущей папке. |
| `git clone <url>` | Клонирование удалённого репозитория на локальный ПК. |
| `git status` | Просмотр состояния файлов (изменённые, добавленные, удалённые). |
| `git add <file>` | Добавление файла в отслеживаемые (stage). |
| `git add .` | Добавление всех изменённых файлов. |
| `git commit -m "message"` | Фиксация изменений с комментарием. |
| `git push origin main` | Отправка изменений в удалённый репозиторий (ветка main). |
| `git pull origin main` | Получение актуальных изменений из удалённого репозитория. |
| `git log --oneline` | Просмотр истории коммитов в сокращённом виде. |
