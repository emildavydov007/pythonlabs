# Python. Лабораторная работа №3
## Рекурсивные и нерекурсивные решения

## Условия задач

1. Напишите две функции для расчёта суммы элементов вложенного списка:
   - вариант с использованием рекурсии;
   - вариант без рекурсии (итеративный).

   Пример:
   ```python
   >>> sum_nested([1, [2, [3, 4, [5]]]])
   15
   ```

2. Реализуйте две функции для вычисления последовательностей $a_k$ и $b_k$ по формулам:

   $$a_k = \frac{1}{2} \left( \sqrt{b_{k-1}} + \frac{1}{2} \sqrt{a_{k-1}} \right), \quad b_k = \frac{3}{2} \sqrt{b_{k-1}} + \frac{1}{2} a_{k-1}^2, \quad a_1 = b_1 = 1$$

   с начальными условиями $a_1 = b_1 = 1$. Один вариант следует выполнить рекурсивно, другой — итеративно.

## Описание проделанной работы

- Разработаны оба варианта каждой функции: рекурсивный и итеративный.
