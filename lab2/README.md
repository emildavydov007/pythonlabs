# Лабораторная работа №2

## Условия задач
1. Ольга составляет таблицу кодовых слов для передачи сообщений, каждому сообщению соответствует своё кодовое слово. В качестве кодовых слов Ольга использует 4-буквенные слова, в которых есть только буквы A, B, C, D, X, Y, Z. При этом первая буква кодового слова — это буква X, Y или Z, а далее в кодовом слове буквы X, Y и Z не встречаются. Сколько различных кодовых слов может использовать Ольга?

2. Значение арифметического выражения  записали в системе счисления с основанием 3. Сколько цифр 2 содержится в этой записи?

3. Найдите все натуральные числа, принадлежащие отрезку \([40000; 50000]\), у которых ровно пять различных **нечётных** делителей (количество чётных делителей может быть любым). Выведите найденные числа в порядке возрастания.
## Описание проделанной работы

### Задача 1
Для решения использован модуль `itertools.product`. Первая буква выбирается из 3 вариантов (X, Y, Z), остальные 3 буквы — из 4 вариантов (A, B, C, D). Функция перебирает все возможные комбинации и возвращает их количество.
> Python
```
from itertools import product

def count_code_words() -> int:
    
    first_letters = ('X', 'Y', 'Z')
    other_letters = ('A', 'B', 'C', 'D')

    count = 0
    for first in first_letters:
        for rest in product(other_letters, repeat=3):
            count += 1
    return count
```
### Задача 2
Вычисляется значение выражения \(9^8 + 3^5 - 9\), затем число переводится в троичную систему счисления путём последовательного деления на 3. Подсчитывается количество цифр 2 в полученной записи.
> Python
```
def count_twos_base3():
    n = 9**8 + 3**5 - 9
    
    # перевод в 3-ичную систему
    base3 = ''
    while n > 0:
        base3 = str(n % 3) + base3
        n //= 3
    
    return base3.count('2')
```
### Задача 3
Для каждого числа из диапазона \([40000; 50000]\):
1. Удаляются все множители 2 (чётные делители не влияют на количество нечётных делителей)
2. Подсчитывается количество делителей оставшегося нечётного числа
3. Если их ровно 5, число добавляется в результат
> Python
```

def find_numbers():
    result = []
    
    for n in range(40000, 50001):
        m = n
        
        # убираем все двойки
        while m % 2 == 0:
            m //= 2
        
        # считаем делители m
        count = 0
        for i in range(1, int(m**0.5) + 1):
            if m % i == 0:
                count += 1
                if i != m // i:
                    count += 1
        
        if count == 5:
            result.append(n)
    
    return result
```
